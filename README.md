### ⌗ دوال وقواعد البيانات المتقدمة (DATABASE METHODS)

// .all() --- جلب كافة البيانات المسجلة داخل المصفوفة المعنية
=> global.db.users.all()

// .get() --- البحث عن سجل مستخدم أو بيانات محددة بواسطة المعرف أو رقم الهاتف (jid, lid, id, _id)
=> global.db.users.get(jid)

// .delete() --- حذف سجل معين من قاعدة البيانات بالاعتماد على الرقم أو المعرف (jid, lid, id, _id)
=> global.db.users.delete(jid)

// .drop() --- مسح وتصفير مصفوفة البيانات بالكامل وحذف محتواها
=> global.db.users.drop() // --- لمسح المصفوفات (Array)
=> global.db.statstic.drop() // --- لمسح الكائنات (Object)

// طريقة الاستدعاء المتسلسل (Chainable Method)
=> global.db.bots(jid)?.data?.users.get(jid)


> [!IMPORTANT]
> لإضافة هيكل بيانات أو جداول جديدة إلى global.db (على سبيل المثال: global.db.reports)، يرجى إضافة هيكل البنية المطلوبة داخل دالة structure() الموجودة في ملف models.js.
>
